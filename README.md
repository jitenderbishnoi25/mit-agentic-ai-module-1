# LLM Token Pricing Comparison 🚀

A modern, glassmorphic web page designed to easily compare the token counting usage and API pricing estimates across various generations of GPT models.

## 📝 Overview
This simple static web application breaks down the average token usage for common expected API calls (10k, 100k, 500k, 1m) and compares the pricing bounds across:
- **Next-Gen Models**: GPT-5.x & O1/3
- **Current/Legacy Standard**: GPT-4 & GPT-3.5
- **Legacy Basic**: GPT-3

## ✨ Features
- **Glassmorphism UI**: Beautiful translucent styling overlaying floating ambient light orbs.
- **Micro-Animations**: Hover interactions that smartly highlight model groups across table rows and spanning cells.
- **Fully Responsive**: Optimised token breakdown view whether you are on mobile, tablet, or desktop.
- **Zero Dependencies**: Pure HTML, CSS, and lightweight vanilla JavaScript. No framework overhead.

## 🛠️ Built With
- **HTML5**: Semantic and accessible document structure.
- **Vanilla CSS3**: Leveraging CSS Variables for theming, backdrop-filters for the glass effect, and complex custom animations.
- **Google Fonts**: Uses `Inter` for highly polished data legibility, and `Outfit` for gorgeous typography and headings.

## 🚀 Getting Started

Simply open or double-click the `index.html` file in any modern web browser to view the page locally.
